CSS3

(Cascading Style Sheets) Es un lenguaje de diseño de páginas web, interpretado por todos los navegadores web gráficos, independientemente del dispositivo o sistema operativo. El HTML le dice al navegador la estructura de la página web, mientras que el CSS le dice como renderizarlo.

Los procesos que otorga CSS3 son excelentes tales como bordes, fondos, color, textos, transformaciones, transiciones, cajas, contenido, opacidad. son elementos esenciales para el diseño y estructura para cualquier página web. El nombre de hojas de estilo en cascada se debe a que los estilos aplicados se heredan de forma Jerárquica, es decir que si aplicamos un font-size:10px; a la etiqueta BODY, todos los elementos de texto de nuestra página tendrán este tamaño de letra a menos que les especifiquemos otro tamaño.

Maquetación Básica
  1. Width: Especifica el ancho que debe tener un elemento.
  2. Height: Especifica el alto que debe tener un elemento.
  3. Margin: Especifica el espacio de separacion que toma el elemento respecto a los demas. Se puede especificar por separado arriba, abajo, izquierda y derecha.
  4. Padding: Especifica un margen interno del objeto el cual aumenta su tamaño, este se puede especificar por separado arriba, abajo, izquierda y derecha.
Texto y fuentes
  1. Font-family: Permite especificar el tipo de letra a usar.
  2. Font-size: Permite especificar el tamaño de la letra.
  3. Font-style: Permite elejir el estilo de letra (normal, cursiva, etc).
  4. text-decoration: Permite dar decoracion al texto como subrayado, tachado, linea superior, etc.
  5. Text-align: Permite centrar, justificar, alinear a la izquierda y derecha el texto.
Color y Fondo
  1. Color: Permite dar un color al elemento, este color se puede indicar mediante palabras preestablecidas como red, blue, etc., mediante combinacion rgb como tambien por medio de valores de color hexadecimales.
  2. Backgroud-color: Permite definir el color de fondo que tenra un objeto, al igual que color puede ser una palabra, una combinacion rgb o un valor hexadecimal.
  3. Backgroud-image: Permite definir una imagen como fondo del elemento.
  4. Background-repeat: Permite determinar como se adapta la imagen de fondo con respecto al tamaño del elemento.
  5. box-shadow: Permite dar una sombra al elemento, esta sombra puede ser interna y externa.
Listas
  1. List-style-image: Permite elegir una imagen como viñeta a una lista.
  2. List-style-type: Permite elegir el tipo de viñeta que usara una lista (circulo, cuadrado, etc).
Bordes
  1. border: Anade un borde al elemento.
  2. border-color: Permte definir el color del borde.
  3. border-style: Permite definir el estilo del borde(solido, punteado, etc).
  4. border-radius: Permite redondear las esquinas de un elemento.

Git - Sistema de Control de Versiones

Git es un sistema de control de versiones creado por Linus Torvalds en el año 2005. Este software nos permite publicar todos los cambios que se registren en nuestros archivos que se están trabajando, de esta forma podemos tener un registro de todas las modificaciones que sufran estos archivos y así en dado caso de algún error poder identificar rápidamente un punto de la versión al cual poder regresar.
